# IMDb Movie Recommendation System

![Python](https://img.shields.io/badge/Python-3.10%2B-blue?logo=python)
![FastAPI](https://img.shields.io/badge/FastAPI-0.100%2B-009688?logo=fastapi)
![scikit-learn](https://img.shields.io/badge/scikit--learn-1.3%2B-F7931E?logo=scikit-learn)
![License](https://img.shields.io/badge/License-MIT-green)

A **content-based movie recommendation system** built on the IMDb Top 1000 dataset from Kaggle.  
The system vectorises movie metadata (genre, director, cast, overview) with **TF-IDF** and ranks candidates by **cosine similarity**.  
A production-ready **FastAPI** service exposes the model via a clean REST API.

---

## Features

| Feature | Details |
|---|---|
| **Content-based filtering** | TF-IDF on genre + director + cast + overview |
| **Fuzzy title matching** | Typos and partial titles are tolerated |
| **FastAPI REST API** | Swagger UI auto-docs at `/docs` |
| **Pagination & filtering** | `/movies?genre=Drama&min_rating=8.5` |
| **Reproducible training** | One-command pipeline: `python src/train.py` |
| **Full test suite** | pytest covering preprocessing, recommender & API |

## Dataset

**IMDb Top 1000** — [Kaggle](https://www.kaggle.com/datasets/harshitshankhdhar/imdb-dataset-of-top-1000-movies-and-tv-shows)

| Column | Description |
|---|---|
| `Series_Title` | Movie title |
| `Released_Year` | Release year |
| `Runtime` | Duration in minutes |
| `Genre` | Comma-separated genres |
| `IMDB_Rating` | IMDb score (0–10) |
| `Meta_score` | Metacritic score |
| `Director` | Director name |
| `Star1–4` | Lead actors |
| `Overview` | Plot summary |
| `Gross` | Box office gross (USD) |

## Quick Start

### 1. Clone & install dependencies

```bash
git clone https://github.com/<your-username>/IMDb-Movie-Recommendation-System.git
cd IMDb-Movie-Recommendation-System

pip install -r requirements.txt
```

### 2. Train the model

```bash
python src/train.py
```

Expected output:
```
Loading raw data …
Cleaning dataset …
Building feature tags …
Fitting TF-IDF + cosine similarity …
Training complete ✓
  vectorizer   → models/vectorizer.pkl
  sim_matrix   → models/similarity_matrix.pkl
```

### 3. Start the API server

```bash
python -m uvicorn api.main:app --reload
```

Open **http://localhost:8000/docs** for interactive Swagger UI.

## API Reference

### `GET /recommend`

Returns top-N content-based recommendations for a given movie.

| Parameter | Type | Default | Description |
|---|---|---|---|
| `title` | string | **required** | Movie title (fuzzy matching supported) |
| `top_n` | int | 10 | Number of recommendations (max 50) |

**Example request:**
```http
GET /recommend?title=Inception&top_n=5
```

**Example response:**
```json
{
  "query": "Inception",
  "matched_title": "Inception",
  "total_results": 5,
  "results": [
    {
      "Series_Title": "Interstellar",
      "Released_Year": 2014,
      "Genre": "Adventure, Drama, Sci-Fi",
      "IMDB_Rating": 8.6,
      "Director": "Christopher Nolan",
      "Overview": "A team of explorers travel through a wormhole ...",
      "Poster_Link": "https://...",
      "similarity_score": 0.4821
    }
  ]
}
```

### `GET /movies`

Paginated catalogue with optional filters.

| Parameter | Default | Description |
|---|---|---|
| `page` | 1 | Page number |
| `page_size` | 50 | Results per page (max 200) |
| `genre` | — | Filter by genre (e.g. `Drama`) |
| `min_rating` | 0.0 | Minimum IMDb rating |

### `GET /movie/{title}`

Full metadata for a single movie.

### `GET /health`

Service health check — confirms model is loaded.

## How It Works

```
Raw CSV
  │
  ▼ clean_dataset()
Cleaned DataFrame  ←── NaN handling, type casting, deduplication
  │
  ▼ build_features()
Feature tags  ←── "genre director star1 star2 star3 star4 overview words"
  │
  ▼ TfidfVectorizer (5000 features, 1-2 ngrams)
TF-IDF matrix  (1000 × 5000)
  │
  ▼ cosine_similarity()
Similarity matrix  (1000 × 1000)
  │
  ▼ MovieRecommender.recommend()
Top-N similar movies  ←── fuzzy title match → sort by score → return
```
